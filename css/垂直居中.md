## 垂直居中

```html
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  display: flex;
  jusitify-content: center;
  align-items: center;
}

.container {
  display: grid;
  jusitify-content: center;
  align-items: center;
}

/** 第三种 start  **/
.container {
  position: relative;
}
.box {
  position: absolute;
  top: 0;
  bottom: 0;
  left: 0;
  right: 0
}
/** 第三种 end  **/

/** 第四种 start  **/
.container {
  position: relative;
}
.box {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: (-50%, -50%);
}
/** 第四种 end  **/

.container {
  display: table;
}
.box {
  display: table-cell;
  verticle-align: middle;
}


.box {
  display: inline-block;  // inline-flex;
  verticle-align: middle;
}
.box::after {
  line-height: 100%;
}

```
